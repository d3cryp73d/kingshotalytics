Patch Notes v1.8: Gear Planner Integration

This update connects the Gear Planner to the Construction Calculator for a more streamlined experience.

New Feature

  • Autofill Buffs: In the Construction Calculator, logged-in users can now check a box to automatically fill the "Total Construction Speed Buff" field with the value from their saved gear sets.
  • This removes the need to manually look up your stats every time you use the calculator.

Note: This feature requires you to have a saved build in the Gear Planner. The calculation is based on placeholder stats until the gear item database is complete.
